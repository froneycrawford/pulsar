Survey Database Information

List of EGRET survey targets:

    The columns in the ASCII file listing correspond to the following:

    Column 01: Arbitrary source index number (SNO)
    Column 02: Source catalog (3EG)
    Column 03: EGRET source name from catalog
    Column 04: Target name in survey
    Column 05: J2000 right ascension (deg.)
    Column 06: J2000 declination (deg.)
    Column 07: Galactic longitude (deg.)
    Column 08: Galactic latitude (deg.)
    Column 09: Maximum expected DM from NE2001 DM-distance model (pc cm-3)

  • ASCII listing of EGRET survey targets sorted by name
  • A Microsoft Excel spreadsheet of EGRET survey targets

File listings of EGRET survey observations:

    The columns in the ASCII file listing correspond to the following:

    Column 01: Source name (EGRET source)
    Column 02: Filename (.pkmb) of pointing for DVD archive
    Column 03: Tape name of original DLT
    Column 04: File number on original DLT tape
    Column 05: Galactic longitude of center beam (deg.)
    Column 06: Galactic latitude of center beam (deg.)
    Column 07: Mean Julian Date
    Column 08: Gregorian date (yymmdd_hhmmss)
    Column 09: Number of blocks in file
    Column 10: Sample time (ms)
    Column 11: Freq of top channel in band (MHz)
    Column 12: Channel width (MHz), it's negative!
    Column 13: Number of beams recorded in observation
    Column 14: Integration time of observation (s)
    Column 15: Notes

  • ASCII listing of EGRET survey observations sorted by EGRET source name
  • ASCII listing of EGRET survey observations sorted by filename (.pkmb)
  • A Microsoft Excel spreadsheet of EGRET survey observations

    Note: the following pointing was not observed in the survey: G1638d.
    Note: the following pointings were observed twice in the survey: G0439Ad, G1314b, G1457d, G1638c, G1652b, G1726a, G1726b, G1726c, G1726d.

Beam position listing of EGRET survey observations:

    The columns in the ASCII beam position listings correspond to the following:

    Column 01: Filename (.pkmb) of beam
    Column 02: J2000 right ascension (hh:mm:ss)
    Column 03: J2000 declination (dd:mm:ss)
    Column 04: Galactic longitude (deg.)
    Column 05: Galactic latitude (deg.)

  • ASCII listing of EGRET survey beam positions sorted by filename (.pkmb)

    Note: as of August 1, 2003 this file lists all 3016 beams observed in the survey (100% of total).

DM trial listing for EGRET survey observations:

    The columns in the ASCII DM trial listings correspond to the following:

    Column 01: DM index of trial (integer)
    Column 02: DM value of trial (pc cm^-3)

  • ASCII listing of DM trials for EGRET survey processing

Raw data DVD archive:

    Requests for raw survey data may be made to Froney Crawford (fcrawfor@fandm.edu).

    Each DVD contains a single survey pointing (in the form of 13 separate beam files) and a README file. The size of each beam file is typically 4102 blocks * 49792 bytes per block = 204246784 bytes (194.8 MB) per beam.

    An example README file looks like this: 

     
Parkes P406 EGRET Pulsar Survey Raw Data DVD Archive
